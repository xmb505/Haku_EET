EET 电梯控制面板 — 操作说明
1. 内召：点击对应楼层按钮，模拟轿厢内选层
2. 外召：点击上下箭头按钮，模拟厅外召唤
3. 门控制：开门 / 关门 / 停止门
4. 系统命令：启动 / 停止 / 急停 / 复位
5. 自定义指令：直接输入 JSON 发送到后端
POST /control — 控制接口
GET /stattrak — 数据接口
